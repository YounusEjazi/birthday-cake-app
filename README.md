## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser

4. Allow camera permissions when prompted

## How to Use

- **Blow out candles**: Open your mouth and blow toward the camera
- **Torch candles**: Point your index finger at a candle flame on the screen

- MediaPipe (Face Mesh & Hands)
